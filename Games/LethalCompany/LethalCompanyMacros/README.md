<h2>Getting Started</h2>

1. Download [AutoHotkey](https://github.com/samfilip/mods)
1. Extract this folder and run lc_macros.exe
1. Your macros should be active while Lethal Company is running.


MACROS

CTRL+K: Shuts down all turrets

CTRL+m: View Monitor

CTRL+t: transmit

Right or Left arrow: Cycle through player cams

CTRL+Backspace: Hit backspace ten times?

CTRL+f: Jumps to the store.
